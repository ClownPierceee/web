---

## Структура проекту

```
bilous-vladyslav/
└── website/
    ├── index.html          — Головна сторінка
    ├── services.html       — Сторінка послуг
    ├── contact.html        — Сторінка контактів з формою
    ├── css/
    │   ├── reset.css       — Скидання стилів браузера
    │   └── styles.css      — Основні стилі сайту
    ├── js/
    │   └── main.js         — JavaScript логіка
    ├── images/             — Зображення сайту
    ├── audio/              — Аудіофайли (ЛР №5)
    └── video/              — Відеофайли (ЛР №5)
```

---

## Технології

- **HTML5** — семантична розмітка (ЛР №1–6)
- **CSS3** — стилізація, Flexbox, Grid (ЛР №7–8, 9–10, 11–12)
- **JavaScript (ES6+)** — валідація форм, DOM-маніпуляції (ЛР №13–14)
- **CSS Media Queries** — адаптивна верстка (ЛР №13–14)
- **Google Fonts** — Playfair Display + Lato

---

## Сторінки та функціонал

### `index.html` — Головна
- Hero-секція з фоновим зображенням
- Секція «Про нас» з CSS Grid і позиціонуванням
- Картки послуг (CSS Flexbox)
- Портфоліо з фільтрацією (JavaScript)
- Відгуки клієнтів
- Форма зворотного зв'язку з JS-валідацією

### `services.html` — Послуги
- Hero заголовок з хлібними крихтами
- Детальні картки послуг (CSS Grid)
- Таблиця порівняння пакетів (HTML5 таблиця, ЛР №3–4)

### `contact.html` — Контакти
- Повна форма замовлення (всі типи полів ЛР №6)
- Картки з контактною інформацією
- Список визначень `<dl>` (ЛР №2)
- Валідація форми засобами JavaScript (ЛР №11–12, 13–14)

## Відповідність лабораторним роботам

| ЛР | Тема | Реалізація у проекті |
|----|------|----------------------|
| ЛР №1 | HTML5 основи | Всі сторінки: h1–h6, p, a, img, br, strong |
| ЛР №2 | Списки HTML5 | nav ul>li, footer ul>li, dl/dt/dd на contact.html |
| ЛР №3–4 | Таблиці HTML5 | Таблиця пакетів на services.html |
| ЛР №5 | Аудіо та відео | Папки audio/ та video/ |
| ЛР №6 | Форми HTML5 | Форми на index.html та contact.html |
| ЛР №7–8 | Основи CSS | styles.css: селектори, background, border-radius, shadow |
| ЛР №9–10 | CSS Flexbox | header, nav, картки, форми |
| ЛР №11–12 | Стилізація форм CSS | form, input, select, textarea стилі |
| ЛР №13–14 | Позиціонування + адаптив | fixed header, absolute img, media queries, JS |

## Breakpoints (адаптивна верстка)

- **360px** — мінімальна ширина (мобільні пристрої)
- **768px** — мобільні / планшети
- **1024px** — планшети / малі десктопи  
- **1200px** — стандартні десктопи
- **1920px** — великі екрани

---

## Запуск проекту

Відкрити файл `index.html` у браузері Google Chrome, Firefox або Edge.

> **Рекомендується:** використовувати розширення **Live Server** у VS Code для коректної роботи.
